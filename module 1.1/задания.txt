Задача 1: Реализуйте класс ThreadSafeQueue - потокобезопасную очередь на основе std::queue, используя мьютекс и условную переменную для синхронизации. Должны быть методы push(), pop(), empty().

Задача 2: Напишите функцию parallel_accumulate, которая вычисляет сумму элементов контейнера, используя несколько потоков. Разделите контейнер на части и суммируйте каждую часть в отдельном потоке.

Задача 3: Реализуйте класс ObjectPool с использованием RAII для управления пулом объектов. Пул должен быть потокобезопасным.

Задача 4: Напишите шаблонную функцию is_container, которая с помощью SFINAE проверяет, является ли тип контейнером (имеет методы begin() и end()).

Задача 5: Создайте простой Dockerfile для сборки C++ проекта с использованием CMake. Проект должен компилироваться с поддержкой C++17.
